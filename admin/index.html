Wilkommen im Admindashboard!

Hier werden Raumgruppen verwaltet, Benutzer eingeladen, Räume hinzugefügt, etc..

Zum Erstellen einer neuen Raumgruppe:

Sie können eine neue Raumgruppe unter dem Menüpunkt "Meine Raumgruppen" -> "Raumgruppe hinzufügen" -> "Neu erstellen" hinzufügen

Zum Beitreten zu einer bestehenden Raumgruppe

Sie können Raumgruppen unter dem Menüpunkt "Meine Raumgruppen" -> "Raumgruppe hinzufügen" -> "Einer Raumgruppe beitreten" beitreten
